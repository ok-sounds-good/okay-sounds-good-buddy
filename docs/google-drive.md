# Google Drive Sharing Guide

Canonical web version: <https://docs.divebar.me/google-drive.html>

This local reference grounds the tests and release behavior. The web guide remains
authoritative.

## Rules Relevant To OKSG

- Release filenames use `Code-#### - Artist - Title` with spaces around the
  separators so karaoke catalog software can parse the fields.
- Keep a creator code consistent and increment the release number for each track.
- Name the CDG and MP3 identically inside a CDG ZIP.
- Use `&` for two credited artists; list multiple credited artists directly rather
  than using a feature-credit abbreviation.
- Put singer-specific notes with the artist and music-specific notes with the
  song title.

The source also provides background on why consistent naming improves the
experience for KJs and karaoke catalog software.
