# Harley's Naming Guide

Canonical web version: <https://docs.divebar.me/harleys-guide.html>

This local reference grounds the tests and release behavior. The web guide remains
authoritative.

## Rules Relevant To OKSG

- Begin at `0001` and increment the release number for every distinct track.
- Keep the release structure as `Code-#### - Artist - Title`.
- Move singer-related notes, such as duet or backing-vocal information, into the
  artist field. Keep music-version notes with the title.
- Capitalize words, while preserving an artist's intentional styling when needed.
- Remove a leading `The` from an artist except for `The Who` and `The The`.
- For titles beginning with `The`, retain two-word titles unchanged; for longer
  titles, move it to the end as `, The`.
- Preserve punctuation except commas, with `, The` as the exception.
- Replace `And`, `Featuring`, and similar feature-credit forms with `&`.
